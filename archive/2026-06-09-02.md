## Key decisions

- Used `requestAnimationFrame` to defer `getBBox()` calls until SVG is rendered in DOM
- White stroke halo on map labels for legibility over coloured regions
- Seeded LCG random (`Math.imul`) ensures reproducible subsets per seed value
- SVG `width`/`height` attributes removed, `width:100%` set — responsive and print-safe
- Color swatches use lighter print-safe shades with `print-color-adjust: exact`
