## Key decisions

- **Natural Earth 50m** for Morocco/Algeria: good balance of coastal detail vs. download
  speed (~1.5 MB total; only 2 features kept). 110m was too coarse; 10m too large (22 MB).
- **`min_lat = 35.3`**: lowest value that keeps Melilla (35.29°N) just inside the SVG.
  With `min_lat = 35.8` (old value), Melilla was off-screen.
- **Geometric clip approach**: `africa_gdf.geometry.union_all().intersection(shapely_box(...))`
  — clip to MAINLAND_BOUNDS so the southern edge of the Africa path is exactly at SVG y=800
  (the map's min_lat line), giving a natural fill from coastline to map edge.
- **Fallback path preserved**: if the Natural Earth download fails, `add_africa_strip()`
  still uses the hand-drawn bezier dome.
- **Stroke on Africa path**: added `#888 / 0.5px` — in the reference PDF the coastline has
  a visible border edge separating Morocco from the map background.
