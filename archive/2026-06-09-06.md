## Key decisions

- **CLAUDE.md approach**: single file + SessionStart hook. The hook injects `additionalContext`
  into the conversation so Claude doesn't forget the confirmation even in non-hook-aware sessions.
- **Canary Islands label fix**: baking transform into coordinates (vs. applying offset in JS).
  Chose baking because it's a one-time change to the SVG source and `getBBox()` then works
  naturally without any special-case code.
  Scale applied: 0.7191, tx=53.01, ty=698.36.
- **Africa strip**: hand-drawn SVG path (not from real GeoJSON) — approximates the coastline
  well enough for a children's worksheet; no dependency on data sources.
- **Print layout**: switched `@page` to `A4 landscape`; used CSS Grid `grid-template-columns: 36mm 1fr`
  to move the worksheet header into a left sidebar, freeing the full page height for the map SVG.
  SVG set to `height: 174mm; width: auto` for predictable scaling.
