## Key decisions

- **Africa fill=none в генераторе**: бесполезно менять вручную после генерации —
  теперь бэкается прямо в `generate_svg.py`. При перегенерации не нужно патчить SVG.
- **getBBox() + transform**: SVG spec: `getBBox()` возвращает bounding box в локальных
  координатах элемента до transform. Для Канарских островов с `translate(10,705)` нужно
  добавлять (tx, ty) к cx/cy. Исправлено в `addLabels()` в index.html.
- **Micro-sliver filter**: порог 1e-4 sq degrees в geographic space. Реальная Сеута ≈ 0.004
  sq deg, slivers ≈ 0.00002 sq deg — разница в 200×, порог безопасен.
- **Португалия в africa-context группе**: CSS `#africa-context path { stroke: #888 }`
  применяется ко всем путям в группе, включая portugal-outline. Стиль одинаковый → правильно.
- **references/ в .gitignore**: сам файл README.txt тоже убран — папка полностью приватная.
